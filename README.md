This project focuses on identifying waterfront transformation potential in industrial inland port-cities in the hinterland of Port of Rotterdam. The methodology adopted in this thesis is mainly based on quantitative urban morphology. As a result, a multi-scalar framework is established to identify the transformation potential by integrating both macro-economic development dynamics and micro-urban morphological imbalances. During this process, R scripts were used as a critical tool for quantitative analysis, promoting transferability and reproducibility.
